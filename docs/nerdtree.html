Open / close

Key / CommandAction
<C-n>Toggle NERDTree
:NERDTreeFindOpen NERDTree and reveal the current file

Navigating the tree

KeyAction
j / kMove down / up
oOpen file or expand/collapse directory
<Enter>Same as o
tOpen in new tab
sOpen in vertical split
iOpen in horizontal split
pJump to parent directory
PJump to root
K / JJump to first / last child of current directory

Changing the root

KeyAction
CSet the directory under cursor as new root
uMove root up one directory
cdChange Vim's working directory to the node under cursor

File operations

KeyAction
mOpen the NERDTree menu (create, rename, delete, copy, move)
aCreate a new file or directory (inside the menu)

Display

KeyAction
IToggle hidden files (dotfiles)
fToggle file filters
FToggle file display (show directories only)
AZoom NERDTree pane to full width / back
?Toggle built-in help
qClose NERDTree
